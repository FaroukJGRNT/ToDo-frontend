# React Application

## Description

This React application consumes an Express.js API connected to a MySQL database.

## Architecture

- **Frontend**: React
- **Backend**: Express.js 
- **Database**: MySQL

## Installation

Install dependencies:

```bash
npm install
```

## Getting Started

Start the application:

```bash
npm run start
```

## Prerequisites

- Node.js
- npm
- MySQL
- Express.js server configured and running
